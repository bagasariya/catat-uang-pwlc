# Sistem Pencatatan Keuangan (catat-uang-pwlc.vercel.app)

## Deskripsi
Aplikasi pencatatan pemasukan dan pengeluaran berbasis web menggunakan Node.js, Express, EJS, dan MySQL.

## Fitur
- Login
- Dashboard
- Pencatatan pemasukan
- Pencatatan pengeluaran
- Laporan transaksi

## Teknologi
- Node.js
- Express.js
- EJS
- MySQL
